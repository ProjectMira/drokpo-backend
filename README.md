# drokpo-backend

Backend for **Drokpo** (གྲོགས་པོ་, "friend"), an app for Tibetans to make friends — nearby and across the diaspora.

- API: FastAPI on Cloud Run, fronted by Firebase Hosting — see [backend/README.md](backend/README.md)
- Push notifications: Cloud Functions in [functions/](functions/)
- Docs: [tech stack](docs/TECH_STACK.md), [data schema](docs/DATA_SCHEMA.md), [chat system](docs/CHAT_SYSTEM.md), [deployment](docs/DEPLOYMENT.md)
